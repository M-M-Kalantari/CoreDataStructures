# CoreDataStructures

A collection of small C++ programs focused on **core data structure concepts**, pointer manipulation, and classic computer science problems.  
This repository is intended for learning, practice, and reference.

  Pointer basics and pointer-related operations.

- **Queue-&-Stack.cpp**  
  Queue and stack implementation and usage.

- **Sum-without-int.cpp**  
  Calculating sum without using integer arithmetic operators.
